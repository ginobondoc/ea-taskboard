One-Time Setup (15 min)

Required once per workspace. Free forever. No credit card needed.
  1. Go to console.firebase.google.com → Create project → name it anything → disable Analytics → Create
  2. Inside the project: Build → Firestore Database → Create database → Start in test mode → Choose region → Enable
  3. Go to Project Settings (⚙) → General → Your apps → Web app (</>) → Register app → copy the firebaseConfig object below
  4. Paste your config in the fields below, then click Continue
